# Виртуальная частные сети - VPN

### Задание:

1. Настроить GRE между офисами Москва и С.-Петербург;
2. Настроить DMVPN между офисами Москва и Чокурдах, Лабытнанги.

2. Настроим DMVMN между Москва и Чокурдах, Лабытнанги;
